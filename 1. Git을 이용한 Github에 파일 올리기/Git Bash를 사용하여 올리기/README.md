
### Git Bash를 사용하여 Github에 올리기

<h2>1. Github에 올리고 싶은 파일 오른쪽 클릭을 해 Git Bash Here선택</h2>

<p>Git Bash Here 선택</p>

<h2>2. 환경설정시 이름과 이메일을 등록과 확인</h2>
이름과 이메일 등록하기 <br>
git config --global user.name 이름 <br>
git config --global user.email 이메일<br>
<br>

제대로 등록되었는지 확인 <br>
git config --list
